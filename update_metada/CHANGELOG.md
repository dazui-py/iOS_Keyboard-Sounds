## iOS Keyboard Sounds

Systemless iOS-style keyboard sounds module for Android.

# Changelog

#### V1.0.0
- Removed obsolete Magisk installer flags from `customize.sh`.
- Removed legacy logic that was not needed for current root solutions.
- Removed unnecessary complexity from installation scripts.
- Simplified boot-time handling to keep the module focused on keyboard sounds only.
- Cleaned up module structure for better compatibility and easier maintenance.
- Reduced redundant code and unused assets.
- Refined the module to match its actual purpose: iOS keyboard sound replacement.
